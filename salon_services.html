Posh Salon: User Manual ✂️

Welcome! This tool helps you quickly see which stylists perform which services, their prices, and any special notes.

Adding & Removing

  • To Add a New Stylist: Type a name in the "New Stylist Name..." box and click the Add Stylist button.
  • To Add a New Service: Type a name in the "New Service Name..." box and click the Add Service button.
  • To Remove a Stylist: Double-click a stylist's name, then click the trash can icon (🗑️) that appears.
  • To Remove a Service: Double-click a service name in the header, then click the trash can icon (🗑️) that appears.

Editing Information

  • To Edit a Service Name: Double-click on any service name in the header.
  • To Edit a Stylist Name: Double-click on any stylist's name in the table.
  • To Change a Stylist's Level: Single-click the level text (e.g., "Level Five") under a stylist's name.
  • To Change a Price: Single-click on any price (or on the "Add Price" text).

Using Sticky Notes

  • To Add a Note: (1) Click Add Note. (2) Click a color. (3) Your cursor becomes a `➕`. Click any square in the table. (4) Type your message and click Save Note.
  • To View a Note: Click the colored square in the corner of any cell.
  • To Delete a Note: Open the note, then click the "Delete this note" text link.

Saving & Managing Your Data

  • Save: Click this to save your work to the current browser.
  • Export: Click this to save a backup file (`salon-data.json`) to your computer.
  • Import: Click this to load data from a file. This will completely replace the current matrix.
