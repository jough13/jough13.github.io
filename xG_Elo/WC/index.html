How the World Cup Hybrid Engine Works

This World Cup Hybrid xG-Elo system recalculates international tournament fixtures, combining actual scorelines with expected goals (xG) metrics to find the true power rankings of international football.

Key World Cup Model Adjustments

1. Neutral Grounds & Designated Host Boost (+25)
Unlike domestic club leagues where every home team gets a point boost, World Cup matches are played on neutral ground. A +25 Home Advantage boost is applied only when a designated host nation plays on home soil.

2. Tournament K-Factor (50)
Because national teams play fewer fixtures and World Cup games represent the pinnacle of football, the K-Factor is set to 50. This makes the model responsive to tournament runs and high-stakes upsets.

3. 70/30 Blended Score with xG Fallback
For matches with xG data (2018–2026), performance is graded 70% on match result and 30% on xG share. For historical tournaments prior to 2018, the model falls back to 100% match scorelines.

4. Penalty Shootouts (0.75 / 0.25)
If a knockout match ends in a draw after 120 minutes, the standard 0.5 draw score is overridden for the shootout. The team that wins on penalties is awarded 0.75, and the loser is awarded 0.25.

5. Inactive Coasting Penalties
When viewing "All-Time Cumulative" rankings, teams that miss a World Cup cycle can be penalized so they do not "coast" on ancient high ratings. Every team in the database is registered at 1500 before the first tournament begins; missing a cycle will retroactively penalize unproven teams. You can toggle a flat point deduction or a 15% regression toward the 1500 baseline for every tournament missed.

The Formulas

1. Expected Performance (E):
E = 1 / (1 + 10^((Rating_Away - (Rating_Home + Host_Boost)) / 400))

2. Blended Actual Performance (A):
A = (0.70 * Match_Result) + (0.30 * xG_Share)
*Match Result = 1 (Win), 0.75 (PK Win), 0.5 (Draw), 0.25 (PK Loss), 0 (Loss)

3. Rating Update:
New_Rating = Old_Rating + [50 * Blowout_Multiplier * (A - E)]
