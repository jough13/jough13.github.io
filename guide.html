Welcome to your new blog! This guide covers everything you need to know to write and format your posts using Markdown.

Your Posting Workflow

Adding a new post is a two-step process:

  1. Create your post file: Write your content in a new file and save it with a `.md` extension inside your `posts/` folder (e.g., `my-new-post.md`).
  2. Update the manifest: Open `posts.json` and add an entry for your new post at the top of the list. Make sure the `"file"` property points to your new `.md` file.

Basic Formatting

Headings

Use the hash symbol (#) to create headings. More hashes mean a smaller heading level.

# This is a Heading 1 (used for post titles automatically) ## This is a Heading 2 ### This is a Heading 3

Text Styles

You can easily make text bold, italic, or create links.

*This text will be italic.* **This text will be bold.** [This is a link to Google](https://www.google.com)

Adding Content

Images

First, place your image file in the assets/images/ folder. Then, use the following syntax in your post:

![A description of the image for accessibility](assets/images/your-image-name.jpg)

Lists

You can create bulleted (unordered) or numbered (ordered) lists.

* An item in a bulleted list * Another item 1. The first item in a numbered list 2. The second item

Blockquotes

To create a blockquote, start the line with a > symbol. This is great for quoting people.

> "The journey of a thousand miles begins with a single step."

Advanced Content

Code Blocks

Showcase code snippets by wrapping them in triple backticks (```). You can specify the programming language for syntax highlighting.
